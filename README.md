# UD25-Spring-REST-ER-Peliculas-Salas

## URIs endpoints: 
```
Peliculas
GET       /api/peliculas
POST      /api/pelicula
GET       /api/pelicula/{id}
PUT       /api/pelicula/{id}
DELETE    /api/pelicula/{id}

Salas
GET       /api/salas
POST      /api/sala
GET       /api/sala/{id}
PUT       /api/sala/{id}
DELETE    /api/sala/{id}
```
